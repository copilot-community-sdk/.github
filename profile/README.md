# GitHub Copilot Community SDKs

Community-maintained SDKs for building extensions and integrations with GitHub Copilot.

> ⚠️ **Disclaimer**: These are unofficial, community-maintained SDKs. They are not officially supported by GitHub and may break due to changes in the Copilot CLI or APIs.

## About

This repository hosts community-driven SDKs that enable developers to build tools and integrations with GitHub Copilot. These SDKs are created and maintained by the community to fill gaps in official tooling and provide language-specific implementations.

## Contributing

Contributions are welcome! If you'd like to add a new SDK or improve existing ones, please open a pull request.
